PySciCalc is a command line scientific calculator written in Python.  
It focuses on clean structure, clear error handling, and unit tested core logic.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Powers and roots (square root and n-th root)
- Trigonometric functions in degrees: sin, cos, tan
- Logarithms: natural log (ln) and base 10 (log10)
- Factorials for non negative integers
- Input validation and helpful error messages
- Modular design with unit tests

## Project structure

pyscicalc/
  main.py
  operations.py
  test_operations.py

## Requirements

Python 3.10 or later
